SCREEN MOVEMENT

  • zz cursor line to center of screen
  • zt cursor line to top of screen
  • zb cursor line to bottom of screen
  • CTRL-e move screen forward one line
  • CTRL-y move screen backward one line
  • CTRL-u move backward half a screen
  • CTRL-d move forward half a screen
  • CTRL-b move backward a full screen
  • CTRL-f move forward a full screen
  • Note: In command mode, ^ means Ctrl

  • CTRL-v start visual-block mode

  • aw highlight/select a word
  • ab/aw a block with {}/()

  • Esc exit visual mode

HOW TO THINK IN VIM

  • To be efficient in VIM, learning commands is not enough. To accomplish editing tasks elegantly, you must learn to think in "VIM VERB NOUN MODIFIER" statements. Because VIM is a symbol language with a formal grammatical structure, these VIM idioms describe what you want to do in a somewhat natural way and are the heart of VIM wizardry.
  • VERBS are the thing you want to do - (change), (delete), (yank), (visual), etc.
  • MODIFIERS change how nouns are understood - (inside), (around), (until), (to), (find), etc.
  • NOUNS are what you are editing - (word), (sentence), (paragraph), (block), etc.
  • Some examples:
  • diw delete inside word to delete word under cursor
  • ciw change inside word, respecting citation
  • ci" change inside quotation marks to change string inside next marks
  • c* change everything from the cursor till the letter x
  • v} visually select (around) this paragraph
  • yi( copy the phrase inside the parentheses
  • dfoo delete from here until the second occurrence of 'foo'
  • fypx then change the occurrence of 'y', move forward 2 words, and change word 'x'
  • h,j,k,l for fine tuning. If you are using them more than a few times, there's probably a more efficient way to move or accomplish what you're doing. Stop and think in sentences!

SEARCH

  • /pattern search for pattern
  • ?pattern search for pattern backwards
  • Search Pattern: Finds...
  • a the character a
  • ^a line beginning with char a
  • [abc] string containing any of a,b or c
  • a.*a string beginning with a char a
  • \a, AA, AAA disable char special meaning
  • n next
  • N previous

REPLACE

  • :s/old/new/ replace next old with new
  • :s/old/new/g replace instances of old with new in current line
  • :%s/old/new/g replace all old with new throughout file
  • :%s/old/new/gc replace all old with new in file with confirmations
  • :%s/old/new/g& replace instances of old with new between lines 1 to 5, inclusive
  • :g/string/d delete lines containing/ not containing instances of string x

CASE

  • ~ switch char case, move forward
  • U,u uppercase/lowercase line
  • gU,g~ uppercase/lowercase word
  • VgU,Vg~ invert case for word/line

MISCELLANEOUS COMMANDS

  • w or ZZ write (save) and quit file
  • w! write file overriding protection
  • q! quit file (discarding any changes)
  • ! command use UNIX command
  • r file read file contents into curr. file
  • :map <key> <C> define keystroke <K> as a command sequence
  • :ab x y abbreviate x as y; when x is typed it insert y instead; x needs to be full words or phrases (3)
  • una[b] a disable abbreviation or mapping
  • :set [all] display options set so far; if all is included, shows default options as well
  • :set o option activate option o; set o? show value of option o
  • :set no[u] deactivate option u
